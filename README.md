<h1>Radial Offhand</h1>
This repository of Minecraft mod which adds an intuitive radial menu for offhand.

## Multiversion Setup
This project is built using [Stonecutter](https://stonecutter.kikugie.dev/) and it's [community template](https://github.com/rotgruengelb/stonecutter-mod-template) created by rotgruengelb.

If you are contributing and want to understand how to use it, check [official documentation](https://stonecutter.kikugie.dev/wiki/start/) of Stonecutter.

In the branches, you can find legacy versions of the mod that previously used a branch-based approach for multi-version support.

## Support version    
Currently supported platforms and versions:
- Fabric, Forge 1.20.1
- Fabric, Neoforge 1.21.1
- Fabric, Neoforge 1.21.10
- Fabric, Neoforge 1.21.11
